\# Project Skeleton



Go ahead and steal this as a starter scaffold for your project. Some of the basics have been stubbed in already. The console stuff is in there.



There are also example gradle files here. Make sure to put in the correct URLs. We don't want to leave them facing the public internet, so they are not included in this repo.



These two gradle files should be enough for IntelliJ to detect and prep your project. You don't actually need to init the directory. IntelliJ will see this is a gradle project and do it for you.
