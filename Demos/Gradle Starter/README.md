\# Gradle Starter



Go ahead and steal this as a starter for Gradle projects. Make sure to put in the correct URLs. We don't want to leave them facing the public internet, so they are not included in this repo.



These two gradle files should be enough for IntelliJ to detect and prep your project. You don't actually need to init the directory. IntelliJ will see this is a gradle project and do it for you on first open.
